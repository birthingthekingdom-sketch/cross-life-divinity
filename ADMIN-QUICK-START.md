## Managing Courses & Assignments

### Adding Assignment Due Dates

1. Go to: `/admin` → Find the course
2. Click "Edit" on any lesson
3. Scroll to "Assignment" section
4. Toggle "This lesson has an assignment"
5. Enter assignment prompt
6. Set due date using the date picker
7. Click "Save"

### Grading Assignments

1. Go to: `/admin/grading`
2. You'll see all pending submissions
3. Click "Grade" on any submission
4. Review the student's work
5. Enter scores for:
   - Theological Understanding
   - Content Quality
   - Writing & Presentation
6. Add feedback comments
7. Click "Submit Grade"
8. Student receives email notification automatically

## Troubleshooting

### Can't Access Admin Dashboard?

1. Make sure you're logged in with birthingthekingdom@gmail.com
2. Clear browser cookies and log in again
3. Go directly to the `/admin` URL
4. Your account is verified and has admin role

### Students Can't Register?

- Registration uses Manus OAuth (the "Continue to Cross Life School of Divinity" screen)
- Students click "Use another account" to create new accounts
- After registration, they must verify their email
- Then they can log in and enter the access code

### Access Code Not Working?

- Check that the code is active: `/admin/access-codes`
- Verify courses are linked to the code
- Make sure student is logged in before entering code
- Student must go to `/enroll` page to enter code
