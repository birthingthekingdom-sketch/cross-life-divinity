## How to Navigate the Materials

### For New Students
1. Start with **README.md** - Main entry point
2. Read **GED_COURSE_CATALOG.md** - Understand program structure
3. Review **resources/time-management/TIME_MANAGEMENT.md** - Create study plan
4. Start with **study-guides/STUDY_GUIDES_INDEX.md** - Begin studying
5. Use **practice-tests/PRACTICE_TEST_GUIDE.md** - Take practice tests

### For Experienced Students
1. Review **resources/test-strategies/TEST_STRATEGIES.md** - Refresh strategies
2. Take **actual-exam/ACTUAL_EXAM_GUIDE.md** - Prepare for real exam
3. Use **resources/reference-materials/REFERENCE_MATERIALS.md** - Review key concepts
4. Take **practice-tests/** - Practice with full-length tests

### For Instructors
1. Review **README.md** - Understand program structure
2. Reference **GED_COURSE_CATALOG.md** - Know all courses
3. Use **study-guides/STUDY_GUIDES_INDEX.md** - Teach concepts
4. Assign **practice-tests/** - Have students practice
5. Reference **resources/** - Provide support materials
