## Production Tips

### 1. Add Pauses for Impact
The [PAUSE] markers in the script are important. They give viewers time to absorb key messages.

In 11 Labs, you can add pauses by:
- Using commas (short pause)
- Using periods (medium pause)
- Using ellipses ... (longer pause)
- Using line breaks (longest pause)

### 2. Emphasize Key Phrases
In 11 Labs, you can emphasize words by:
- **Bold text** = slightly more emphasis
- *Italic text* = softer, more emotional
- ALL CAPS = strong emphasis (use sparingly)

### 3. Background Music
Use royalty-free music from:
- **Epidemic Sound** (subscription)
- **Artlist** (subscription)
- **YouTube Audio Library** (free)
- **Uppbeat** (free with attribution)

**Music style:** Inspirational, uplifting, corporate, cinematic

**Volume:** Keep music at 20-30% volume so voiceover is clear

### 4. Video Length
- **60 seconds** = Best for social media (Facebook, Instagram, YouTube ads)
- **90 seconds** = Good for website homepage
- **120 seconds** = Full introduction for YouTube channel

### 5. Export Settings
- **Resolution:** 1080p (1920x1080) minimum, 4K if possible
- **Format:** MP4 (H.264 codec)
- **Aspect Ratio:** 
  - 16:9 for YouTube, website
  - 1:1 (square) for Facebook/Instagram feed
  - 9:16 (vertical) for Instagram Stories/Reels
