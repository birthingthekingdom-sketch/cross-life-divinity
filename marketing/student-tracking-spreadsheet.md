**Purpose:** Track your progress toward 50 students and identify what's working

**Tool:** Google Sheets (free) or Excel

**Columns Explained:**

- **Date:** When they enrolled
- **Name:** Student name
- **Source:** How they found you (see source categories below)

- **Amount:** How much they paid
- **Payment Method:** Full payment or payment plan

## Source Categories

Track where each student came from:

1. **Personal Text** - Direct text message from you
2. **Personal Call** - Phone call from you
3. **Alumni Email** - Email to past students
4. **Church Announcement** - Sunday announcement
5. **Facebook Post** - Organic Facebook post
6. **Facebook Live** - Facebook Live Q&A
7. **Facebook Group** - Posted in FB group
8. **Referral** - Referred by another student
9. **Church Partnership** - Church sponsorship program
10. **Webinar** - Free webinar attendee
11. **Website** - Found you through Google/website
12. **Other** - Specify in notes

**Why track source?**
- See what's working
- Double down on successful channels
- Stop wasting time on channels that don't convert

## Dashboard Summary (Top of Sheet 1)

**Create a visual dashboard at the top:**

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│           50 STUDENTS GOAL TRACKER                     │
│                                                         │
│   Current Students: 30                                 │
│   Goal: 50                                             │
│   Remaining: 20                                        │
│                                                         │
│   Progress: [████████████░░░░░░░░] 60%               │
│                                                         │
│   Total Revenue: $9,750                                │
│   Average per Student: $325                            │
│                                                         │
│   Top Source: Facebook Live (8 students)               │
│   Best Week: Week 4 (12 students)                      │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**Use formulas to auto-calculate:**
- Current student count: `=COUNTA(B2:B51)` (counts rows with names)
- Remaining: `=50-[Current Students]`
- Progress %: `=[Current Students]/50`
- Total Revenue: `=SUM(E2:E51)`
- Average per Student: `=[Total Revenue]/[Current Students]`

## How to Use This Spreadsheet

### Daily (5 minutes)

1. **Add new enrollments** to Sheet 1
2. **Update outreach tracker** (Sheet 3) with today's activities
3. **Check follow-up column** - who needs contact today?

### Weekly (15 minutes)

1. **Update weekly progress** (Sheet 2)
2. **Review what worked** - which sources got the most students?
3. **Plan next week** - double down on what's working
4. **Pay referral bonuses** (Sheet 7)

### Monthly (30 minutes)

1. **Review revenue** (Sheet 4)
2. **Analyze trends** - which months were strongest?
3. **Adjust strategy** - what needs to change?
4. **Celebrate wins!** - you're making progress!

## Pro Tips

### 1. Update Daily

**Don't wait until the end of the week.**

Update it every day (takes 5 minutes).

Why? You'll forget details if you wait.

---

### 2. Use Conditional Formatting

**Make it visual:**

- **Green** = Enrolled
- **Yellow** = Thinking about it
- **Red** = Not interested

**How:**
- Select cells
- Format → Conditional formatting
- Set rules based on status

---

### 3. Set Reminders

**Use the "Next Action" column to set reminders:**

- "Follow up 12/20" → Set phone reminder for 12/20
- Don't rely on memory - use the system

---

### 4. Celebrate Milestones

**Add a "Milestones" section:**

- ✅ 10 students - 12/18
- ✅ 25 students - 12/28
- ⬜ 50 students - ?

**Celebrate each milestone!**

- Post on Facebook
- Thank your supporters
- Treat yourself to something nice

---

### 5. Share Progress

**Every week, post an update:**

"Week 2 update: 15 students enrolled! 35 more to go. Thank you to everyone who's supporting this journey!"

**Why?**
- Builds momentum
- Encourages more enrollments
- Creates social proof

## Sample Weekly Review Questions

**Every Sunday, ask yourself:**
