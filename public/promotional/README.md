## Brand Guidelines

**Colors:**
- Primary Blue: #1e3a8a
- Accent Gold: #f59e0b
- White: #ffffff

**Logo:**
- Golden lion with cross emblem

**Typography:**
- Headlines: Georgia, serif (or Merriweather)
- Body: Open Sans, sans-serif

## Usage Instructions

### Social Media Posts

1. **Facebook**: Use `social-media-facebook.png` for posts and ads. Pair with enrollment links and course highlights.

2. **Instagram**: Use `social-media-instagram.png` for feed posts. Add carousel posts with individual course highlights using the people-focused images.

3. **Twitter/X**: Use `social-media-twitter.png` as profile header. Share individual promotional images with course announcements.

### Email Marketing

1. **Welcome Emails**: Use `email-template-welcome.html` and replace placeholders:
   - `{{LOGO_URL}}` - Link to your hosted logo
   - `{{STUDENT_NAME}}` - Student's name
   - `{{DASHBOARD_URL}}` - Link to student dashboard

2. **Enrollment Confirmations**: Use `email-template-course-enrollment.html` and replace:
   - `{{COURSE_CODE}}`, `{{COURSE_TITLE}}`, `{{COURSE_DESCRIPTION}}`
   - `{{CPD_HOURS}}`, `{{LESSON_COUNT}}`
   - `{{COURSE_URL}}`, `{{DASHBOARD_URL}}`

### Print Materials

- Print `Cross-Life-Divinity-Promotional-Flyer.pdf` on high-quality paper
- Recommended: Glossy finish, full color
- Distribute at ministry conferences, churches, and theological events

### Website Banners

Use any of the landscape promotional images as website hero banners or featured content sections.

## File Formats

## Customization

To customize email templates:
1. Open HTML files in a text editor
2. Replace placeholder text (marked with `{{}}`)
3. Update colors in inline styles if needed
4. Test in email client before sending
